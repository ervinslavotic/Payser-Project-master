Requirements

The Google App engine allows us the developers to develop applications on Google's Infrstructure
so in order for the software to run we need SDK's and componants with the Google App Engine for Python.
The Python environment is very productive for GAE since it provides rich Python APIs for Google Datastore,
Google Accounts and URL fetch and email services componants.

Jinja & Gmail

Payser is available for two types of users - Employee and Employer.
Payser is based and hosted via the Google App Engine.
This provides Payser with a framework to manage user data efficiently.
Payser holds a description of the resources which it uses to function

correctly. For templates we use Jinja which is an environment
based within our Python Scripts to load our templates corresponding
to the applications link.
